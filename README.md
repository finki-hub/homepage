## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:8080](http://localhost:8080) to view it in the browser.

### Build for Production

```bash
npm run build
```

The build artifacts will be generated in the `dist` folder.

### Preview Production Build

```bash
npm run preview
```
